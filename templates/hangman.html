Hangman

More Info

This is a simple Hangman game I created using Tkinter, a graphical user interface library in Python. The goal of the game is for the user to guess what word the program is thinking of. As the user guesses letters, the blanks in the word will appear if the guess is correct. If it is wrong, the user will “lose a life”. They have a total of 6 lives to guess the word before they lose the whole game.

I originally coded this in Python, but I also wanted to understand GUIs more. I learned about Tkinter through some research and was able to grasp onto it fairly easily. I then later converted my python code into something that was more user friendly and visually pleasing.
